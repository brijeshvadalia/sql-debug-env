Advanced capabilities

Everything an RL agent needs

Every observation is information-rich. Every step returns a gradient signal.

🧠
Multi-Turn Memory
Every observation includes the last 5 steps — SQL query, error message, reward, result count, query complexity label. Agents learn from failed attempts within the same episode.
conversation_history
📊
Query Complexity Classifier
Automatically classifies every submitted query as simple / moderate / complex / advanced — detecting JOINs, subqueries, window functions, CTEs, aggregations.
query_complexity
⚡
EXPLAIN Performance Metrics
Full EXPLAIN QUERY PLAN summary with scan count, index usage, speedup ratio vs baseline, and efficiency score. Hard tasks reward faster queries.
performance_metrics
💡
3-Level Hint System
Progressive hints with configurable reward penalty. Level 1 gives direction, Level 2 identifies the bug, Level 3 gives the exact fix. Each costs 10% (max 30%).
POST /hint
📈
Episode Analytics
When done=True, every response includes an EpisodeSummary: best_reward, improvement_rate, termination_reason, step_rewards array, and hint penalty breakdown.
episode_summary
🎓
Curriculum Learning
Auto-advances easy → medium → hard → expert when agent achieves avg ≥ 0.8 over 3 episodes. Full mastery progress tracking per task.
GET /curriculum
